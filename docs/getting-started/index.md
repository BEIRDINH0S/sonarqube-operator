# Getting Started

Three pages to get you productive:

1. [**Concepts**](concepts.md) — vocabulary you need (CRD, reconcile loop, finalizer, drift detection)
2. [**Installation**](installation.md) — install the operator on your cluster (Helm or `kubectl apply`)
3. [**Quick Start**](quick-start.md) — deploy your first SonarQube instance and a project, end to end

Read them in order if SonarQube-on-Kubernetes is new to you. Skip to the Quick
Start if you already know what an operator is and just want to see results.
