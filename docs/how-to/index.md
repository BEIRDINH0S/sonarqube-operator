# How-To Guides

Task-oriented recipes for common operations. Pick the one matching what you're
trying to do.

| Guide | Use when… |
|---|---|
| [Install Plugins](install-plugins.md) | You need to add language analyzers or third-party plugins |
| [Manage Projects](manage-projects.md) | You want projects to be created and configured by Git, not by hand |
| [Quality Gates as Code](quality-gates-as-code.md) | You want every project to inherit the same quality gate, with drift correction |
| [Manage Users and Groups](manage-users-and-groups.md) | You want declarative user provisioning instead of manual UI clicks |
| [Token Rotation](token-rotation.md) | You need CI tokens that rotate automatically on a schedule |
| [High Availability](high-availability.md) | You're running the operator with multiple replicas and need to understand leader election |
